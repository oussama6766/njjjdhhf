# Ramadan Prep — دليل الإعداد

## 1. إعداد Supabase
1. افتح [supabase.com](https://supabase.com) وادخل لمشروعك
2. اذهب لـ **SQL Editor**
3. الصق محتوى ملف `supabase-setup.sql` وشغّله

## 2. تشغيل المشروع (في Terminal)
```bash
cd "C:\Users\baych\OneDrive\Desktop\cf"
npm install
npm run dev
```
المشروع سيشتغل على: http://localhost:5173

## 3. البناء (للنشر على Vercel)
```bash
npm run build
```
بعدها ارفع مجلد `dist` على Vercel.
